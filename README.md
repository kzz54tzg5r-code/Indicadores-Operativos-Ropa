## Menú tipo carrusel del video

Se sustituyó el `st.radio` por un carrusel HTML real:

- La pestaña seleccionada queda centrada.
- Las tarjetas anterior y siguiente quedan parcialmente visibles.
- Se puede deslizar horizontalmente con el dedo.
- No aparecen círculos de radio ni el texto “Pestañas”.
- La tarjeta activa es blanca, con texto azul y línea rosa.
- Las demás tarjetas quedan sobre fondo azul degradado.
- Al seleccionar un reporte permanece en la misma ventana.
- En computadora conserva una barra horizontal compacta.
